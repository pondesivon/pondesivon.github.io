Boyutlandırma Alternatifleri - CSS

CSS ile boyutlandırma yaparken kullanılabilecek stiller aşağıda verilmiştir.

Kod

/* Dörtlü Boyutlandırma */
.sinif{ padding:2px/*üst*/  4px/*sağ*/  6px/*alt*/  8px/*sol*/ }

/* Üçlü Boyutlandırma */
.sinif{ padding:2px/*üst*/  4px/*sol ve sağ*/  6px/*alt*/ }

/* İkili Boyutlandırma */
.sinif{ padding:2px/*üst ve alt*/  4px/*sağ ve sol*/ }

/*Tekli Boyutlandırma*/
.sinif{ padding:2px/*üst, alt, sağ ve sol*/ }
