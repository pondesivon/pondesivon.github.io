Option Compare Binary Ve Option Compare Text Kullanımı - Microsoft Excel

Microsoft Excel VBA bünyesinde isteğe bağlı olarak kullanılabilen Option Compare Binary ifadesinin ne işe yaradığına kısaca bakalım.

Bilgi

  • Option Compare Binary

    Option Compare Binary varsayılan olarak açık bir şekilde gelir. Yani hiç yazılmadığında da bu seçenekle çalışır.

    Eğer Option Compare Binary ifadesi General Declaration kısmına yazılırsa string ifadeler büyük - küçük harf ayrımı yaparak karşılaştırma yapar.

  • Option Compare Text

    Eğer Option Compare Text ibaresi General Declaration kısmına yazılırsa string ifadelerde büyük küçük harf ayrımı yapmadan karşılaştırma yapar.


Kod

Aşağıdaki kod ile iki seçeneği karşılaştırabilir ve iki seçeneğin nasıl bir sonuç verdiğini koddaki şart kısmı ile öğrenebilirsiniz.


Option Explicit
Option Compare Text

Sub StringKontrol()
    If ("metin" = "Metin") Then
        MsgBox "String ifadeler eşit."
    Else
        MsgBox "String ifadeler eşit değil."
    End If
End Sub
