input Value Değerini Ayarla - Javascript

Javascript input etiketi için id değerine göre value değerini ayarlama işlemi nasıl yapılır görelim.

Kod

<input type="text" id="metin">

<script type="text/javascript">
    var element = document.getElementById("metin");
    element.value = "Degeri buraya yaziyoruz.";
</script>
