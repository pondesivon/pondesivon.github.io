Diziyi forEach Döngüsü İle Yazdır - Javascript

Javascript ile forEach döngüsü kullanarak diziyi yazdırma işlemi yapan alternatif bir javascript kodudur.

Dosyalar

index.html


<!doctype html>
<html lang="tr-TR">
<head>
    <meta charset="UTF-8">
    <title>Javascript</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>

    <script type="text/javascript" src="dosya.js"></script>
</body>
</html>

dosya.js


var numbers = [5, 10, 50, 100, 150, 200];

numbers.forEach(function(entry) {
    document.write(entry + "<br>");
});
