Hakkında

Web sitesi hakkında çeşitli bilgilerin yer aldığı bilgilendirme sayfası.

Web Sitesinin Tanımı

Bu web sitesi şu şekilde tanımlanmıştır: Ağırlıklı olarak programlama, webmaster ve bilgisayar konularındaki içeriklerin paylaşıldığı kişisel veri tabanı. Alternatif not defteri.

Ön Bilgilendirme
  • Bu alan adını veya bu alan adı altındaki herhangi bir sayfayı, içeriği vs. ziyaret ettiğiniz zaman aşağıda belirtilen şartlara, durumları vs. kabul ettiğiniz varsayılır.
  • Bu sayfadaki bilgiler önceden haber vermeksizin değiştirilebilir, güncellenebilir, kaldırılabilir.
  • Bu web sitesindeki içerikler önceden haber vermeksizin değiştirilebilir veya kaldırılabilir.
  • Eğer aşağıda belirtilen şartları, durumları kabul etmiyorsanız, lütfen bu web sitesini kullanmayınız.
Web Sitesinin Amacı

Bu web sitesi, araştırılan konulardan üretilen sonuç içeriğin yeniden kullanılabilecek şekilde saklandığı bir not defteridir. Bu web sitesinin genel amaçları aşağıdaki maddelerle özetlenebilir.

  • Atıl şekilde duran notların daha ulaşılabilir olmasını sağlamak.
  • Dijital ortamda bir referans noktası oluşturmak.
Sorumluluk Reddi

Bu web sitesinde yayınlanan içerikler kontrol edilerek yayınlanmaktadır fakat yine de sitedeki hiçbir içerik için hiçbir şekilde bir garanti verilmemektedir. Bu yüzden de bu web sitesindeki içerikleri kullanarak yapacağınız şeyler tamamıyla sizin kontrolünüz altındadır. Bu web sitesindeki içerikleri kullanarak yapılanlardan herhangi bir şekilde web sitesi sahibi sorumlu tutulamaz. Ziyaretçi web sitesine girdiği andan itibaren bu şartları kabul etmiş olduğu varsayılır.
