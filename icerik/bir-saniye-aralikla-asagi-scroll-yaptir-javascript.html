Bir Saniye Aralıkla Aşağı Scroll Yaptır - Javascript

Twitter gibi sayfanın sonuna gelindiğinde hemen son gönderinin altından itibaren bir veri grubu daha ekleyerek çalışan siteler için işe yarayabilecek bir javascript kodudur.

Kod


//--------------------------------------------------
//Çalıştır.
//--------------------------------------------------
var indir = setInterval(function(){ asagiScroll() }, 1000);

function asagiScroll() {
    window.scrollTo(0, document.body.scrollHeight);
}

//--------------------------------------------------
//Durdur.
//--------------------------------------------------
clearInterval(indir);

//--------------------------------------------------
//Not
//--------------------------------------------------
//Çalıştırma ve durdurma kodlarını ayrı ayrı çalıştır.
