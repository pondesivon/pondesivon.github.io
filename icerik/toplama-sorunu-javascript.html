Toplama Sorunu - Javascript

Javascript ile toplama işlemi yaptırmaya çalışıyordum fakat bana iki sayıyı birleştirip veriyordu. Sorun meğerse bir dönüşüm problemi imiş.

Senaryo

Javascript ile basit bir toplama işlemi yaptırmaya çalışıyordum, örneğin 3 + 2 = 5 sonucunu döndürsün istiyordum fakat bana 32 sonucunu döndürüyordu.

Çözüm


/* String gibi algılıyor ve birleştiriyor. */
var sonuc = sayi1 + sayi2;

/* İki sayıyı topluyor. */
var sonuc = parseInt(sayi1) + parseInt(sayi2);
