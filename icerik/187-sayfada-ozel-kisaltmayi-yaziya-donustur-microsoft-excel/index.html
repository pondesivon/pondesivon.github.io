Sayfada Özel Kısaltmayı Yazıya Dönüştür - Microsoft Excel

Kendimize özel olarak oluşturduğumuz kısaltmaları A1 hücresine girdiğimizde otomatik olarak normal versiyonunu yazdıracak. Tıpkı mobil tarayıcılarda tanımladığımız/tanımlı kısaltmalar gibi. Örneğin G yazınca Google’a gidiyor.

Kod

Option Explicit

'--------------------------------------------------
'Kısaltmaları ben ":kisaltma" biçiminde düşündüm.
'Siz isterseniz sayı girin, harf girin, kelime girin.
'Başta iki nokta da kullanmayabilirsiniz. Ben kısaltma
'olduğu belli olsun düşüncesiyle hareket ettim.
'--------------------------------------------------
Private Sub Worksheet_Change(ByVal Target As Excel.Range)
    If IsEmpty(Target) Then Exit Sub
    If Target = ":adb" Then Target = "Adem BADEM - BADEMYETİŞTİRENGİLLER"
    If Target = ":çekov" Then Target = "Çekosyovakyalılaştıramadıklarımızdan mıymışsınız?"
End Sub
