Seçilen Alanı Renklendir - Microsoft Excel

Bir hücreyi ya da alanı seçtikten sonra arkaplan rengini değiştiren örnek bir VBA kodudur. Ctrl ile yapılan çoklu seçimler için de renklendirme işlemi yapılıyor.

Ekran Görüntüsü
Kod

Option Explicit

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    Application.ScreenUpdating = False
    
    Cells.Interior.ColorIndex = 0
    Target.Interior.ColorIndex = 8
    
    Application.ScreenUpdating = True
End Sub
Kod (Açıklamalı)

Option Explicit

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    
    '--------------------------------------------------

End Sub
