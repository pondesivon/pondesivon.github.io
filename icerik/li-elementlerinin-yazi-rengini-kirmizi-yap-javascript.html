li Elementlerinin Yazı Rengini Kırmızı Yap - Javascript

Verilen class değerine sahip elementin içindeki li elementlerinin yazı rengini kırmızı yapan javascript kodu alternatifidir.

Dosyalar

index.html


<!doctype html>
<html lang="tr">
<head>
	<meta charset="UTF-8">
	<title>Liste Grupları</title>
</head>
<body>
	<ul class="liste">
		<li>Elma</li>
		<li>Armut</li>
		<li>Kiraz</li>
	</ul>
	<ul>
		<li>Domates</li>
		<li>Biber</li>
		<li>Patlıcan</li>
	</ul>
</body>
</html>

Kod


var listeOgeleri = document.querySelectorAll('.liste li');

for (let i = 0; i < listeOgeleri.length; i++) {
  listeOgeleri[i].style.color = "red";
}
