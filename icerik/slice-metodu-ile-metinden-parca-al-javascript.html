slice Metodu İle Metinden Parça Al - Javascript

slice metodunu ile metinden parça alma alma işlemini yapan javascript kodu alternatifidir.

Kod

var dizi = ["12 Oca 2015 tarihinde yayınlandı", "18 Oca 2015 tarihinde yayınlandı", "30 Oca 2015 tarihinde yayınlandı"];

for (var i = 0; i < dizi.length; i++) {
  dizi[i] = dizi[i].slice(0, 11);
	alert(dizi[i]);
}
