Kelime Sayısına Göre İçerik Göster - WordPress

Bir içerik parçasını ana içeriğin uzunluğuna göre göstermek için kullanılabilecek alternatif bir PHP kodudur.

Kod

Aşağıdaki kod gönderideki kelimelerin sayısını vermektedir.


function word_count() {
	$content = get_post_field( 'post_content', $post->ID );
	$word_count = str_word_count( strip_tags( $content ) );
	return $word_count;
}
Kullanım

Gösterilecek içerik parçasının ana içeriğin altında gösterileceğini varsayalım. Bu durumda aşağıdaki gibi bir kullanım yapılabilir.


<?php the_content(); ?>
<?php if(word_count() >= 200){ ?>
	<hr />
	<!-- İçerik -->
<?php } ?>
