Sayının Üssünü Al - CSharp

Recursive yapı ile sayının üssünü alan alternatif bir C# kodudur.

Kod

double SayininUssunuAl(double sayi, int us)
{
    if (us == 0)
        return 1;
    else if (us == 1)
        return sayi;
    else
        return SayininUssunuAl(sayi, us - 1) * sayi;
}
