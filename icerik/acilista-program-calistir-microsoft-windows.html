Açılışta Program Çalıştır - Microsoft Windows

Bir programı Windows açıldığı zaman çalıştırmak için kullanılabilecek alternatif bir çözümdür.

İşlemler

Kısayol oluştur.

Çalışmasını istediğiniz programın kısayolunu oluşturun.

Çalıştır penceresini aç.

Win + R klavye kısayolunu kullanarak Çalıştır penceresini açın.

Klasör Aç

Çalıştır penceresindeki metin kutusuna %AppData% yazın ve ilgili klasör açılsın.

“Başlangıç” Klasörünü Bul

Klasör açıldıktan sonra sırayla aşağıdaki klasör isimlerini aça aça devam edin.

TR: Microsoft\Windows\Başlat Menüsü\Programlar\Başlangıç

EN: Microsoft\Windows\Start Menu\Programs\Startup

Kısayolu Klasöre Ekle

Başlangıç klasörünü açtıktan sonra artık size nasıl uygun ise oluşturduğunuz kısayolu bu klasöre kopyalayın veya taşıyın.
