Scroll Yapılabilir Elementin Sonuna Git - Javascript

Eğer bir element kaydırma çubuğu ile kaydırılabilir bir özelliğe sahipse bu element için aşağı ilerletme özelliğini yapmaya yarayan javascript kodu alternatifidir.

Kod

//--------------------------------------------------
// Class
//--------------------------------------------------
function scrollYapClass() {
    var x = document.getElementsByClassName("sinif-adi")[0];

    //Elementin sonuna git.
    x.scrollTop += x.scrollHeight;

    //Sayı vererek aşağı git.
    //x.scrollTop += 100;        
}

//--------------------------------------------------
// Id
//--------------------------------------------------
function scrollYapId() {
    var x = document.getElementById("id-adi");

    //Elementin sonuna git.
    x.scrollTop += x.scrollHeight;

    //Sayı vererek aşağı git.
    //x.scrollTop += 100;
}
