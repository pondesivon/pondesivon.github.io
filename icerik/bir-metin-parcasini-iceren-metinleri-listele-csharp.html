Bir Metin Parçasını İçeren Metinleri Listele - CSharp

Verilen metin parçasını içeren metinleri listelemeye yarayan csharp kodudur.

Kod

public static string BirMetinParcasiniIcerenMetinleriListele(string metin, string aranan)
{
    StringBuilder sbd = new StringBuilder();

    MatchCollection mc = Regex.Matches(metin, aranan);

    foreach (Match m in mc)
    {
        sbd.AppendLine(m.Captures[0].ToString());
    }

    return sbd.ToString();
}
