Menü Üstüne Gelindiğinde DropDown Aç, DropDown'dan Ayrıldığında Gizle - jQuery

Menü üzerine gelindiğinde tıklamadan dropdown menüyü açan, mouse imleci dropdown menü alanından çıktığında da dropdown menüyü gizleyen jquery kodudur.

Kod


<div class="dropdown bg-dark">
  <!----------------------------------------------------
  --Bu alana gelindiğinde gizli konumda olan dropdown 
  --menü otomatik olarak açılıyor.
  ---------------------------------------------------->
  <a onmouseenter="$('#icerik-dropdown-ust').show();" id="dropdownMenuLink" class="dropdown">İçerik...</a>
  
  <!----------------------------------------------------
  --Mouse imleci bu div alanından çıktığı zaman dropdown
  --otomatik olarak gizleniyor.
  ---------------------------------------------------->
  <div onmouseleave="$('#icerik-dropdown-ust').hide();" id="icerik-dropdown-ust" class="dropdown-menu">
	<a class="dropdown-item" href="#" target="_blank">İçerik Ekle</a>
	<a class="dropdown-item" href="#" target="_blank">lt gt Dönüştürücü</a>
	<a class="dropdown-item" href="#" target="_blank">Dosya Yükle</a>
	<a class="dropdown-item" href="#">Sorgu</a>
	<a class="dropdown-item" href="#">Hakkında</a>
  </div>
</div>

Etiketler
jquery bootstrap bootstrap 4.4.1
