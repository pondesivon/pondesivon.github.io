Fonksiyon Oluştur Ve Kullan - Javascript

Javascript dosyası üzerine yazılan fonksiyonu html dosyasında nasıl kullanabileceğimizi gösteren bir örnektir.

Dosyalar

index.html


<!DOCTYPE HTML>
<html lang="tr-TR">
<head>
    <meta charset="utf-8">
    <title>Javascript Fonksiyon Çalıştır</title>
    <script type="text/javascript" src="javascript_dosya.js"></script>
</head>

<!--
body etiketinin onLoad olayı için fonksiyonu çalıştırıyoruz.
Yani sayfa yüklenirken çalışacak.
-->
<body onLoad="IlkFonksiyon()">
    Deneme
</body>
</html>

dosya.js

function IlkFonksiyon() {
    document.write("Fonksiyon ile ekrana yazı yazdırma işlemi yapıldı.");
}
