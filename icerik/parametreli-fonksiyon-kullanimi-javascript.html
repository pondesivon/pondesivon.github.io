Parametreli Fonksiyon Kullanımı - Javascript

HTML form elemanlarını kullanarak iki sayı girişi yapacağız ve butona bastığımızda bu iki sayı toplanacak, fonksiyonu da form elemanlarına göre biçimlendireceğiz.

Dosyalar

index.html


<!DOCTYPE HTML>
<html lang="tr-TR">
<head>
    <meta charset="utf-8">
    <title>Javascript İki Sayıyı Topla</title>
    <script type="text/javascript" src="dosya.js"></script>
</head>
<body>
    <form action="" method="post" name="frm_topla">
        Sayı1: <input type="text" name="sayi1" value="15" /><br />
        Sayı2: <input type="text" name="sayi2" value="20" />
        <button type="submit" value="topla" onClick="IkiSayiyiTopla()">İki Sayıyı Topla</button>
    </form>
</body>
</html>

dosya.js


function IkiSayiyiTopla(p1, p2) {
    var p1  = document.frm_topla.sayi1.value;
    var p2  = document.frm_topla.sayi2.value;
    var snc = parseInt(p1)+parseInt(p2);

    return alert(snc);
}
