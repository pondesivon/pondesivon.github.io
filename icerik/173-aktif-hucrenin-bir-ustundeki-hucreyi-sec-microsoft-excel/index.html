Aktif Hücrenin Bir Üstündeki Hücreyi Seç - Microsoft Excel

Enter tuşu varsayılan olarak bir satır altta bulunan hücreyi seçiyor. Bunun tam tersi şekilde çalışan bir makro örneğidir. Bu arada Enter tuşu davranışına müdahale etmiyoruz. Sub yordam sadece.

Kod

Option Explicit

Sub BirUsttekiHucreyeGit()
    If (ActiveCell.Row <> 1) Then
        ActiveCell.Offset(-1, 0).Select
    End If
End Sub
Kod (Açıklamalı)

Option Explicit

Sub BirUsttekiHucreyeGit()

    '--------------------------------------------------
    'Satır numarası 1 olduğunda Offset kodu hata verecek
    'çünkü 0. satır yok. Bu yüzden if koşulunu ekledik.
    '--------------------------------------------------
    If (ActiveCell.Row <> 1) Then
    
        '--------------------------------------------------
        'Aktif hücrenin bir satır üstündeki hücreyi seç.
        '--------------------------------------------------
        ActiveCell.Offset(-1, 0).Select
    End If
End Sub
