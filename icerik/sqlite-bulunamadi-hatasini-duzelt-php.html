SQLite Bulunamadı Hatasını Düzelt - PHP

""fatal error: uncaught error: class "sqlite3" not found in windows" hatasının çözümüne yönelik alternatif bir PHP düzenlemesidir.

İşlemler

php.ini Dosyasını Aç

Gerekli değişiklikleri yapabilmek için php.ini dosyasını aç.

"Dynamic Extensions" Bölümünü Bul Ve Uzantı Ekle

Bu açıklamanın altına ilgili eklemeyi yapacağız. 


;;;;;;;;;;;;;;;;;;;;;;
; Dynamic Extensions ;

;
; Notes for Windows environments :

;   extension folders as well as the separate PECL DLL download.
