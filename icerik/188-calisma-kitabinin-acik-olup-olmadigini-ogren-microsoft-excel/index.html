Çalışma Kitabının Açık Olup Olmadığını Öğren - Microsoft Excel

İsmi ve uzantısı verilen çalışma kitabının (örnek isim olarak "ornek.xlsm" dosyası düşünülebilir) açık olup olmadığını kontrol ediyor ve buna göre mantıksal bir sonuç üretiyor.

Kod

Option Explicit

Function CalismaKitabininAcikOlupOlmadiginiOgren(calismaKitabiIsim As String) As Boolean
    On Error Resume Next
    
    Dim calismaKitabi As Workbook
    Set calismaKitabi = Workbooks(calismaKitabiIsim)
    
    If Err = 0 Then
        CalismaKitabininAcikOlupOlmadiginiOgren = True
    Else
        CalismaKitabininAcikOlupOlmadiginiOgren = False
    End If
End Function
