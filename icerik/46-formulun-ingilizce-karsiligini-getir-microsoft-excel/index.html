Formülün İngilizce Karşılığını Getir - Microsoft Excel

Hem bir formülün, hem de hazırlamış olduğunuz en az 5 satır süren formül grubunun ingilizce karşılığını yazdıran, oldukça kullanışlı olduğunu düşündüğüm bir koddur.

Kod

Option Explicit

Function FormulunIngilizceKarsiliginiGetir(hucre As Range) As String
    FormulunIngilizceKarsiliginiGetir = hucre.FormulaArray
End Function
Kullanım
Formüller A sütununda olsun. B sütununda da KTF'yi kullanacağız.
