"Merhaba Dünya!" Yazdır - jQuery

Sayfa yüklendiğinde sayfaya "Merhaba Dünya!" yazdıran jQuery kodudur.

Kod

index.html


<!DOCTYPE html>
<html>
<head>
	<title>"Merhaba Dünya!" Yazdır - jQuery</title>
	<meta charset="utf-8" />
	<script type="text/javascript" src="jquery.js"></script>
	<script type="text/javascript" src="script.js"></script>
</head>

<!----------------------------------------------------
index.php dosyası tamamen yüklendiği zaman 
merhabaDunya() fonksiyonu çalışacak.
---------------------------------------------------->
<body onLoad="merhabaDunya()">
	
</body>
</html>

script.js


function merhabaDunya() {
	$(document).ready(
		function() {
			document.write("Merhaba Dünya!");
		});
}

Kod (Açıklamalı)

document nedir?

document kelimesi html dosyasının tamamını temsil ediyor.

$(document).ready() nedir?

DOM yüklendikten sonra hangi olayları çalıştıracağımızı belirtmek için kullanıyoruz. Başka bir deyişle tetiklenen olayların çalışabilmesi için bu kod üzerinden kod yazmak gerekiyor.

$() ifadesi nedir?

  • Bu aslında jquery ifadesini temsil ediyor.
  • Yani biz jquery() yazmak yerine $() yazıyoruz.
  • Bu kısaltma Javascript kütüphaneleri içinde başka dillerde de kullanılabiliyormuş.
  • Ayrıca kendi notasyonumuzu da oluşturmak mümkün.

İsimsiz function(){ } ne işe yarıyor?

$(document).ready(function(){ }); kodunun içinde isimsiz bir function(){ } kodu var, bu nedir?

$(document).ready(); kod parçasının parametresiymiş. Gerekli bir parametre, çünkü dosya yüklendikten sonra hangi olayları çalıştıracağımızı bu parametrenin içine yazıyoruz.
