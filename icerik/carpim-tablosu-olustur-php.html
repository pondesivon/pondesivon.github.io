Çarpım Tablosu Oluştur - PHP

PHP - HTML ile yapılmış bir çarpım tablosu örneğidir.

Kod

<!doctype html> 
<html>
	<head>
		<meta charset=utf-8> 
		<title>
			PHP - Çarpım Tablosu Oluştur
		</title>
		<style type="text/css">
			.tablo-stil {
				font-family: 'Consolas', 'Courier New';
				border: 1px dotted; #666;
			}

			.satir {
				padding: 10px;
				border: 1px dotted #666;
			}

			.satir:hover {
				background-color: #cdcdcd;
			}
		</style>
	</head>
	
	<body>
		<h1>Çarpım Tablosu</h1>
		<?php
			echo "<table class='tablo-stil'>";
			for ($i=1; $i<=10; $i++)
			{
				echo "<tr>";
				for ($j=1; $j<=10; $j++)
				{
					echo "<td class='satir'>" . "   $j x $i = " . $j * $i . "  <br>" . "</td>";
				}
				echo "</tr>";
			}
			echo "</table>";
		?>
	</body>
<html>
