Elemente Tıklandığında Verilen Metni Kopyala - Javascript

Elemente tıklanıldığında verilen elementi kopyalamaya yarayan javascript kodudur.

Kod


function metinKopyala(metin) {
	navigator.clipboard.writeText(metin);
}



<span onclick="metinKopyala('Merhaba!')">Verilen Metni Kopyala</span>
