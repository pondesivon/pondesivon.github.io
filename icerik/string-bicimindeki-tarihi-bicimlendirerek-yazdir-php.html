String Biçimindeki Tarihi Biçimlendirerek Yazdır - PHP

String biçimindeki tarih metnini tarih formatında biçimlendirerek yazdıran PHP kodudur.

Kod


<?php 
   $timestamp = strtotime('2023-04-12 18:14:13'); 
   $tarih = date('d F Y • H:i:s', $timestamp);
   echo $tarih;
?>
