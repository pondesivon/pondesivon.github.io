Verilen Class’a Sahip Elementlere strong Etiketi Ekle - Javascript

getElementsByClassName ile aldığımız değerleri bir değişkene aktardığımız ve bu dizi değişkenindeki değerlere for döngüsüyle müdahale ettiğimiz basit bir javascript örneğidir.

Dosyalar

index.html


<!DOCTYPE html>
<html>
<head>
    <title></title>
    <meta charset="utf-8" />
</head>
<body>
    <ul>
        <li class="lst">Elma</li>
        <li class="lst">Armut</li>
        <li class="lst-o">Kuru Fasulye</li>
        <li class="lst-o">Nohut</li>
        <li class="lst">Kivi</li>
        <li class="lst-o">Pirinç</li>
        <li class="lst">Şeftali</li>
        <li class="lst-o">Bulgur</li>
    </ul>

    <script src="dosya.js" type="text/javascript"></script>
</body>
</html>

dosya.js


var listeOge = document.getElementsByClassName("lst");

for (var i = 0; i < listeOge.length; i++) {
    listeOge[i].innerHTML = "<strong>" 
                            + listeOge[i].innerText
                            + "</strong>";
}
